# What is FileStation

You can do

- Update files
- Give others files link
- Files server

# Install

## Install from GitHub

Git clone

```bash
git clone https://github.com/langong-dev/FileStation.git
```

Test server

  Turn into your website link, like `https://langong-dev.github.io/FileStation/`

  click `管理`

Enjoy!

# Use

- `./` Home page
- `?type=admin` Config page
- `?type=view` View files
